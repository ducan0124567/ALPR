# ALPR
Nhận diện biển số xe Việt Nam

## Các thư viện chính
* Keras
* Imutils
* Opencv4
* Numpy
* Matplotlib
* Scipy

## Hướng dẫn cài đặt và chạy chương trình
- Clone source code

```git clone https://github.com/ducan0124567/ALPR```
- Cài đặt các thư viện cần thiết

```pip install -r requirements.txt```
- Chạy GUI
